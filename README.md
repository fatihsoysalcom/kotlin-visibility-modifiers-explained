# Kotlin Visibility Modifiers Explained

This Kotlin example demonstrates the four visibility modifiers: public, protected, internal, and private. It shows how each modifier restricts access to class members within different scopes like classes, subclasses, modules, and files.

## Language

`kotlin`

## How to Run

1. Ensure you have Kotlin installed.
2. Save the code as VisibilityDemo.kt.
3. Compile and run using: kotlinc VisibilityDemo.kt -include-runtime -d VisibilityDemo.jar && java -jar VisibilityDemo.jar
